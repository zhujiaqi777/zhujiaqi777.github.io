2.标识符

程序中用来命名的符号

python中标识符的要求：所有的标识符是由字母，数字，下划线组成，

并且不能由数字开头

格式：变量名 = 值

3.缩进

缩进是指每一行代码前面的空格

python中每行代码前缩进的个数是有规范的，不能随便缩进，

也不能在该缩进的时候没有缩进

4.关键字

关键字是指系统保留下来的具有特殊意义的标识符

5.常用数据类型

int-整型

float-浮点型

bool-布尔型 值只有True(真)和False(假)两个

list-列表 dict-字典 set-集合 tuple-元组，迭代器，生成器，function等

6.两个常用的内置函数：print(),input()

######输出函数-print 　在控制台输出括号内的内容

######输入函数-input　从输入设备上获取输入的内容到程序中

格式 ： 输入结果 = input(提示信息)

######注意：当程序执行到input函数时，程序会停下来等待输入完成才会接着往下执行

##变量

1.什么是变量

变量是用来在内存中开辟空间保存数据的

2.怎么声明变量

####说明：

#####a.变量名-标识符，并且不能是关键字 （要求）

#####见名知意（看到变量名大概知道这个变量中存储的是什么数据）（规范）
例：姓名可以用name表示

#####所有字母都小写，如果变量名由多个单词组成，用下划线隔开 （规范）
例：学生姓名可以表示为student_name

#####b. ‘=’ 赋值符号，将右边的值赋值给左边的变量

#####c. 值：可以是不同类型的数据，也可以是已经声明过的变量或者运算表达式

###3.同时声明多个变量

###4.python中声明变量和给变量赋值的基本原理：
先将数据存起来，再根据数据大小分配相应的内存空间
然后再将变量指向对应的内存区域

###5.内存单位

1mb 1024kb = 1mb
